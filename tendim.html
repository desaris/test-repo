Отчет «Отчет по ГК» - в отчет попадают все лоты со статусом заявки "выигран", с возможностью фильтра по ТК (тендерной компании).

Отчет «Отчет по участию» - в отчет попадают все лоты со статусом заявок: подбор, расчет, принятие решения, участвуем. Должна быть возможность фильтрации по ТК (тендерным компаниям). Группировка дате, сортировка по возрастанию даты подач (от старых к новым).

Отчет «Отчет по торгам» - В отчет попадают все лоты со статусом заявки "Участвуем". Группировка по дате. Сортировка по возрастанию даты (от старых к новым).
